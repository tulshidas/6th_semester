Aglets SDK Readme

We assume the archive was unpacked to directory INSTALLDIR.

Quick'n'Dirty

To get the server running quickly, but insecurely:

Windows
INSTALLDIR\bin\agletsd.bat
Unix
INSTALLDIR/bin/agletsd

Quick'n'Dirty Daemon

To get the server running quickly, but insecurely as a daemon:

Starting the daemon:

Windows
INSTALLDIR\bin\agletsd.bat -daemon
Unix
INSTALLDIR/bin/agletsd -daemon

Starting the daemon's controller:

Windows
INSTALLDIR\bin\daemoncontrol.bat
Unix
INSTALLDIR/bin/daemoncontrol

Tool help

Both agletsd[.bat] and daemoncontrol[.bat] will display usage info when launched with argument -help.

Once launched, daemoncontrol[.bat] supports the online command help.
